# firstlight_lattice — rooms index

The bundled library: 20 prompt "rooms," each a structured prompt for one kind of work.
firstlight routes an incoming request to the room it most resembles. Use `--rooms PATH`
to swap in your own library instead.

| room | what it's for |
|------|----------------|
| `artifact_builder_room_v1` | Build the right artifact in the right form for the right reality level. |
| `artifact_review_room_generic` | the model or editor reviewing an artifact. |
| `auditory_room_v6_canonical` | Improve comprehension in the ear without corrupting meaning on the page. |
| `boundary_room_v1` | Help a no land as a clear, kind, undefended limit — not softened into a false maybe, not hardened into needless coldness. |
| `charge_meter_room_v1` | Run the charge_meter honestly. |
| `debug_room_v1` | Find why something is actually broken and fix the cause, not the symptom. |
| `engineering_scrum_room_v8_2` | Improve the room artifact itself without losing its soul. |
| `generic_opportunity_room_v7` | Find the smallest honest commercial move that creates real proof. |
| `gtm_criticism_room_v2` | the operator / model performing the commercial criticism pass. |
| `gtm_worldview_launch_room_v1` | Protect truth, aliveness, and long-arc integrity while selecting the smallest launch motion that can generate real market signal. |
| `hunting_room_v1` | Let the model chase a scent without knowing where it leads. |
| `life_os_room_v1` | Help the user see what is actually here, separate fact from story, identify the real branch, choose the smallest faithful next move, and emit a durable artifact. |
| `multipart_response_compression_room_v4_3` | First reconstruct the multipart whole faithfully. |
| `opportunity_gtm_room_v5_2` | Generate the most decision-useful likely future branches based on distinct founder behaviors, then recommend the branch and next actions most likely to move the founder toward truthful growth, commercial traction, stronger market position, deeper product understanding, and higher-quality future opportunities. |
| `question_asking_room_v4_consolidated-1__2_` | Help the operator ask the truest useful question, see the live branches clearly, and choose the next move, experiment, or question most likely to increase reality contact without betraying what is alive. |
| `reflection_room_v1` | Help the person reach contact with what is actually true for them right now. |
| `room_builder_room_v2` | Build the right room in the right form for the right reality level. |
| `strategic_room_v1` | Get the most truthful available read of reality and convert it into a decision-useful next move — protecting what is alive without flattering fantasy, protecting realism without rewarding deadness. |
| `teacher_room_v1` | Make it clear to THIS learner at THIS level, and move them one real step. |
| `triage_room_v1` | Find what actually comes first when everything is competing, and name the cuts honestly. |

*20 rooms. Descriptions are each room's own `governing_intent`. The full prompt for any room is in `rooms/<name>.yml`.*
